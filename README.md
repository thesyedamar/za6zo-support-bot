# Za6zo Support Bot

Za6zo Support Bot is a Python customer-support automation project with API routes, a chat widget, WhatsApp integration structure, and a knowledge base for service, pricing, and FAQ responses.

## Highlights

- Support bot architecture for customer questions
- Knowledge base files for services, pricing, passenger FAQs, and driver FAQs
- API routes for chat and WhatsApp-style messaging workflows
- Dockerfile included for deployment preparation
- Test file included for bot behavior validation

## Tech Stack
- Python
- FastAPI-style API structure
- RAG/chatbot components
- HTML chat widget
- Docker

## Run Locally

```bash
pip install -r requirements.txt
python -m api.main
```

## Project Focus

This project demonstrates backend automation for support operations: organizing company knowledge, exposing chat endpoints, and preparing the bot for real customer-facing use.
